## Overview
Forza Horizon Spotify Radio is a popular mod that integrates Spotify directly into Forza Horizon as a fully functional in-game radio station. It replaces the default radio with Spotify branding, allowing you to play your personal playlists and favorite tracks while driving. The mod supports Spotify Connect so you can control music from your phone, desktop app, or browser. It includes a convenient local dashboard and seamless integration with Streamer Mode. Perfect for players who want a personalized and immersive music experience in Forza Horizon 5 and 6.

## Installation
### For Regular Users
1. Download and extract the archive
2. Run the mod installer as Administrator
3. Follow the setup guide
4. Enable Streamer Mode in Forza Horizon
5. Connect your Spotify Premium account
6. Select Spotify Radio in-game and enjoy your music

## Troubleshooting
| Problem | Solution |
|--------------------------------|-------------------------------------------------------|
| Mod doesn't load | Run as Administrator and disable antivirus temporarily |
| Spotify not connecting | Ensure Spotify Premium is active and restart apps |
| No sound in radio | Check audio settings and Streamer Mode |
| Dashboard not opening | Open localhost in browser |
| Game crashes | Verify game files and update the mod |
| Spotify Connect issues | Reinstall mod and restart Spotify |
| Windows 11 compatibility error | Update Windows and graphics drivers |
| Radio station not appearing | Disable Radio DJ and reinstall mod |
| Download / update failed | Use the latest release and check your internet connection |
